Open Source

Fluid Project

Fluid is a CNCF (Cloud Native Computing Foundation) Sandbox project that provides dataset abstraction and elastic acceleration for cloud-native deep learning training jobs.

  • Role: Community Chair
  • GitHub: github.com/fluid-cloudnative/fluid
  • Award: First Prize, China Open Source Innovation Competition (2022)

Fluid enables efficient data access and caching for AI/ML workloads on Kubernetes, improving training efficiency and resource utilization.

Alluxio Project

Alluxio (formerly Tachyon) is a popular open-source data orchestration platform for analytics and AI.

  • Role: Founding PMC Member & Maintainer
  • GitHub: github.com/Alluxio/alluxio

Alluxio provides a unified data access layer that connects compute frameworks to storage systems, enabling high-performance data access for big data and AI workloads.

Apache Spark Contributions
